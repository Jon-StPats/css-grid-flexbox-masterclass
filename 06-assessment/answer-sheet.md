# Theory Quiz Answer Sheet

1. CSS Grid is used to create two-dimensional layouts using rows and columns.
2. Flexbox is used to arrange items in one direction, either a row or a column.
3. `display: grid;`
4. `display: flex;`
5. Grid controls both rows and columns, so it is better for full page layouts.
6. Flexbox makes it easy to align and space navigation links in a row or column.
7. Responsive design means a website adapts to different screen sizes and devices.
8. A media query applies CSS only when certain conditions are met, such as screen width.
9. Examples: `justify-content`, `align-items`, `gap`, `flex-direction`.
10. Examples: `grid-template-columns`, `grid-template-areas`, `grid-area`, `gap`.
