# Marking Guide: Grid and Flexbox Practical

Total: 10 marks

| Criteria | Marks |
|---|---:|
| Grid used for the overall page layout | 2 |
| Grid areas, columns, rows, or gap used correctly | 1 |
| Flexbox used for navigation | 2 |
| Flexbox used for a component such as buttons or cards | 1 |
| Media query included | 1 |
| Mobile layout works clearly | 1 |
| Code is organised, readable, and indented | 1 |
| Student explanation correctly identifies Grid and Flexbox use | 1 |

## Grade Boundaries

| Grade | Marks |
|---|---:|
| Not Achieved | 0-3 |
| Achieved | 4-6 |
| Merit | 7-8 |
| Excellence | 9-10 |

## Teacher Notes

Look for students who understand the reason for using each layout model:

- Grid should control the page sections.
- Flexbox should control one-dimensional alignment inside components.
