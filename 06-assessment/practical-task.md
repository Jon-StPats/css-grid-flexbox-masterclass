# Practical Task: Grid and Flexbox Webpage

Total: 10 marks

## Task

Build a responsive webpage that uses both CSS Grid and CSS Flexbox.

## Required Page Sections

- Header
- Navigation
- Main content
- Sidebar
- Footer

## Technical Requirements

1. Use Grid for the overall layout.
2. Use Flexbox for the navigation.
3. Use Flexbox for at least one button group or component.
4. Add a media query for mobile screens.
5. Keep code organised and readable.

## Evidence

Submit:

- `index.html`
- `style.css`
- A short explanation of where Grid and Flexbox were used
