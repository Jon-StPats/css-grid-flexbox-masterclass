# CSS Grid Challenge

## Task

Modify the Grid layout so the page has:

- Header
- Left sidebar
- Main content
- Right sidebar
- Footer

## Success Criteria

- The page uses `display: grid`.
- The page has at least three columns.
- Grid areas are used correctly.
- The layout is easy to understand from the CSS.

## Extension

Add a media query so the layout becomes one column on small screens.
