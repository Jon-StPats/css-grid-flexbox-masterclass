# CSS Flexbox Challenge

## Task

Create a responsive navigation menu using Flexbox.

The navigation should:

- Display links in a row on desktop.
- Use spacing between links.
- Stack links vertically on smaller screens.

## Success Criteria

- The navigation uses `display: flex`.
- The links are aligned using Flexbox properties.
- The design responds to smaller screens.

## Extension

Add a card or button section that also uses Flexbox.
