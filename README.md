# CSS Grid and Flexbox Masterclass

A short teaching repo for senior web development students.

## Big Idea

Use **CSS Grid** for primary two-dimensional page layouts and **CSS Flexbox** for one-dimensional component alignment.

> Grid for the page. Flexbox for the pieces.

## Learning Objectives

By the end of this mini-unit, students should be able to:

- Explain the difference between Grid and Flexbox.
- Use Grid to create a webpage layout.
- Use Flexbox to align navigation links, buttons, and content inside components.
- Combine Grid and Flexbox in a responsive webpage.
- Use media queries to adapt a page for mobile screens.

## Suggested Lesson Sequence

1. CSS Grid basics
2. CSS Flexbox basics
3. Combining Grid and Flexbox
4. Real-world layout examples
5. Student project
6. Knowledge and practical assessment

## Repository Sections

- `01-grid-basics` - simple Grid layout example and challenge
- `02-flexbox-basics` - simple Flexbox navigation and component example
- `03-combining-grid-and-flexbox` - full page layout combining both
- `04-real-world-layouts` - example layout ideas
- `05-student-project` - starter files for students
- `06-assessment` - theory quiz, practical task, and marking guide
- `teacher-solutions` - completed example solution
