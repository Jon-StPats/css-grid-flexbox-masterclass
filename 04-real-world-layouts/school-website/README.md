# school website Layout

Use this folder as a prompt for students to plan and build a real-world layout.

## Student Questions

1. Which parts of this layout should use CSS Grid?
2. Which parts should use CSS Flexbox?
3. How should the layout change on mobile?
4. How does the layout support usability and accessibility?

## Suggested Approach

- Use Grid for the main page structure.
- Use Flexbox for navigation, buttons, filters, or small groups of content.
- Add a media query for mobile screens.
